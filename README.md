# AI PET(VET) - ระบบบริหารจัดการคลินิกสัตวแพทย์

ซอฟต์แวร์บริหารจัดการคลินิกสัตวแพทย์แบบครบวงจรที่ขับเคลื่อนด้วย AI สร้างขึ้นด้วย Next.js, TypeScript และ Tailwind CSS ระบบนี้ช่วยให้สัตวแพทย์สามารถจัดการสัตว์เลี้ยง เจ้าของ การนัดหมาย รายงานทางการแพทย์ และเข้าถึงความช่วยเหลือจากสัตวแพทย์ที่ขับเคลื่อนด้วย AI ได้

## 🐾 คุณสมบัติ

### 🔐 การตรวจสอบสิทธิ์และความปลอดภัย
- **ระบบเข้าสู่ระบบที่ปลอดภัย**: การตรวจสอบสิทธิ์แบบมืออาชีพด้วยอีเมล/รหัสผ่าน
- **เส้นทางที่ได้รับการปกป้อง**: หน้าเว็บที่สำคัญทั้งหมดต้องมีการตรวจสอบสิทธิ์
- **การจัดการเซสชัน**: สถานะการเข้าสู่ระบบแบบถาวรด้วยคุกกี้ที่ปลอดภัย
- **การเข้าถึงตามบทบาท**: บทบาทสัตวแพทย์ ผู้ดูแลระบบ เจ้าหน้าที่ และพนักงานต้อนรับ

### 🎨 ส่วนติดต่อผู้ใช้ที่ทันสมัย
- **การนำทางแถบด้านข้างแนวตั้ง**: รูปแบบแถบด้านข้างซ้ายแบบมืออาชีพที่ปรับให้เหมาะสมสำหรับขั้นตอนการทำงานของสัตวแพทย์
- **การออกแบบที่ตอบสนอง**: การออกแบบที่เน้นมือถือเป็นหลัก พร้อมแถบด้านข้างที่พับได้สำหรับอุปกรณ์เคลื่อนที่
- **แดชบอร์ดที่สะอาดตา**: รูปแบบการ์ดที่ทันสมัย ​​พร้อมสถิติและการดำเนินการด่วน
- **การนำทางที่สม่ำเสมอ**: แถบด้านข้างที่เป็นหนึ่งเดียวในทุกหน้าเว็บที่ได้รับการปกป้อง

### การจัดการหลัก
- **การจัดการสัตว์เลี้ยง**: บันทึกข้อมูลสัตว์เลี้ยงอย่างครบถ้วน พร้อมชนิด สายพันธุ์ ประวัติทางการแพทย์ การฉีดวัคซีน และข้อมูลเจ้าของ
- **การจัดการเจ้าของ**: ติดตามเจ้าของสัตว์เลี้ยง พร้อมรายละเอียดการติดต่อ และสัตว์เลี้ยงหลายตัว ต่อเจ้าของ
- **การนัดหมาย**: ระบบจองนัดหมายขั้นสูง พร้อมการจัดการช่วงเวลาสำหรับการตรวจสุขภาพ การฉีดวัคซีน การผ่าตัด การดูแลขน และอื่นๆ
- **รายงานทางการแพทย์**: ระบบรายงานที่ครอบคลุมสำหรับผลการตรวจทางห้องปฏิบัติการ การวินิจฉัย บันทึกการรักษา และบันทึกการฉีดวัคซีน

### ความช่วยเหลือที่ขับเคลื่อนด้วย AI
- **ผู้ช่วย AI ด้านสัตวแพทย์**: แชทบอทอัจฉริยะสำหรับการวิเคราะห์อาการ คำแนะนำในการรักษา และการวิจัยทางสัตวแพทย์
- **คำแนะนำอัจฉริยะ**: ข้อมูลเชิงลึกที่ขับเคลื่อนด้วย AI สำหรับการดูแลสัตว์เลี้ยงและการวางแผนการรักษา
- **การตรวจสอบปฏิกิริยาระหว่างยา**: การตรวจสอบความปลอดภัยของยาสำหรับสัตว์เลี้ยงโดยอัตโนมัติ
- **การวิเคราะห์ภาพทางการแพทย์**: การวิเคราะห์ภาพเอ็กซ์เรย์ อัลตราซาวนด์ และภาพวินิจฉัยที่ขับเคลื่อนด้วย AI
- **การประเมินความเสี่ยง**: การทำนายความเสี่ยงด้านสุขภาพสำหรับสัตว์เลี้ยงโดยใช้ AI

### ประสบการณ์ผู้ใช้
- **การนำทางที่ใช้งานง่าย**: แถบด้านข้างแนวตั้งพร้อมไอคอนและป้ายกำกับที่ชัดเจน
- **การดำเนินการด่วน**: เข้าถึงการเพิ่มสัตว์เลี้ยง กำหนดเวลานัดหมาย และสร้างรายงานได้ง่าย
- **มือถือ** **ปรับแต่งแล้ว**: ดีไซน์ที่ตอบสนองต่อทุกอุปกรณ์
- **รองรับหลายภาษา**: พร้อมสำหรับการใช้งานในระดับสากล

## 🚀 เริ่มต้นใช้งาน

### ข้อกำหนดเบื้องต้น
- Node.js 18+
- npm หรือ yarn
- ฐานข้อมูล MongoDB (ในเครื่องหรือบนคลาวด์)

### การติดตั้ง

1. **โคลน repository**
```bash
git clone <repository-url>
cd ai-vet
```

2. **ติดตั้ง dependency**
```bash
npm install
```

3. **กำหนดค่าตัวแปรสภาพแวดล้อม**
```bash
cp .env.local.example .env.local
```
แก้ไขไฟล์ `.env.local` และเพิ่มสตริงการเชื่อมต่อ MongoDB ของคุณ:

MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/ai-vet
